Stack Diagnostics

Tests every layer — API config, Apps Script, Drive, Sheets, and index.html wiring.

Apps Script URL

Paste your Web App URL from Apps Script (the same one that goes in api.js). Leave blank to test only local checks.

1 — api.js is loaded
Checks that api.js is on the page and the API object exists.
2 — APPS_SCRIPT_URL is set
Checks the URL field above and the value inside api.js.
3 — Apps Script is reachable (ping)
Sends a GET ?action=ping to your web app and expects {ok:true}.
4 — Google Sheet is readable (list)
Calls ?action=list and checks that the Sheet returns an array.
5 — Drive upload token works
Calls ?action=token — Apps Script must return an OAuth token and folder ID.
6 — Sheet is writable (add + delete test row)
Adds a test document row, verifies it appears in list, then deletes it. No real files touched.
7 — index.html is wired to API
Inspects index.html to check if dummy data has been replaced with real API calls.
